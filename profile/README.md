# ProdIPData

ProdIPData publishes curated IP geolocation and network intelligence datasets for developers, analysts, DBAs, and data engineering teams.

Our public platform is available at: https://geoiplocations.com

## What we publish

- Monthly IP geolocation datasets
- CSV, Parquet, and MMDB distribution formats
- ASN and registry enrichment
- Country, RIR, bogon, CIDR, port, and root-zone reference catalogs
- Release metadata and documentation
